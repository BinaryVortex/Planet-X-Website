## Getting started

To view the site locally:

1. Clone the repository:

   git clone https://github.com/BinaryVortex/Planet-X-Website.git

2. Open `index.html` in your browser (double-click or use your editor's preview).

Optional: serve it with a local web server for a more accurate environment:

- Python 3: `python -m http.server 8000`
- Node (http-server): `npx http-server .`

Then open http://localhost:8000 in your browser.
